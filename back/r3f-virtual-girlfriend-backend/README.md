## Setup
Create a `.env` file at the root of the repository with your **OpenAI** and **Azure Speech** credentials:

```
OPENAI_API_KEY=sk-...
AZURE_SPEECH_KEY=...
AZURE_SPEECH_REGION=westeurope
AZURE_SPEECH_VOICE=es-ES-ElviraNeural
AZURE_SPEECH_STYLE=general
```

Start the development server with
```
yarn
yarn dev
```
